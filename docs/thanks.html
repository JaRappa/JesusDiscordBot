Quick Start

Jesus Bot will automatically monitor messages for curse words and unkind sentiment. When it detects negativity, it will respond with compassionate, Jesus-inspired messages to remind your community to be kind.

No additional setup is required - the bot is ready to work immediately!

Frequently Asked Questions

How does Jesus Bot know if a message is unkind?

Jesus Bot uses advanced AI from OpenAI to analyze message sentiment and context. It combines this with a curse word filter for comprehensive detection.

Will the bot spam my channel?

No! Jesus Bot has built-in cooldown systems to prevent spam while ensuring it responds to genuinely unkind behavior.

Can I configure the bot's behavior?

Yes! You can adjust sensitivity levels, response probability, cooldown duration, and more through the settings command.

Is the bot free to use?

Yes! Jesus Bot is completely free. We only charge for the API calls OpenAI processes, which we handle.

Where can I get support?

Join our support Discord server or reach out on GitHub for help and feature requests.

Does the bot respect privacy?

Absolutely. We don't store message content - we only analyze messages in real-time to detect negativity.
